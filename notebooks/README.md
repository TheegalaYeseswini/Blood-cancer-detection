These notebooks are the training references used for the inference code in `src/load_models.py`.

- `blood_cancer.ipynb`
  Broad 4-class EfficientNet-B0 classifier for `LEUKEMIA`, `LYMPHOMA`, `MYELOMA`, and `HEALTHY`
- `lukemia_sub.ipynb`
  Leukemia subtype EfficientNet-B0 classifier for `ALL`, `AML`, `CLL`, and `CML`
- `lymphoma_sub.ipynb`
  Lymphoma subtype DenseNet121 classifier for `CLL`, `FL`, and `MCL`
